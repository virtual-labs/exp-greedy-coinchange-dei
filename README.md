### About the Experiment 

This experiment presents an interactive simulation of the Coin Change Problem, demonstrating the working of the Greedy Algorithm through step-by-step visualization. It allows users to input custom amounts and coin denominations, observe algorithm execution via pseudocode highlighting, and analyze decision-making at each step, helping learners understand how greedy strategies solve optimization problems.
